# AI故事视频提示生成器

这是一个简单的静态网站项目，用来生成 AI 故事视频提示词。

项目只使用 HTML、CSS 和 JavaScript，没有使用 React、Next.js、后端、API、数据库或登录功能。

## 功能

- 输入故事主题
- 选择发布平台
- 选择视频时长
- 选择故事风格
- 点击按钮生成提示词
- 显示生成结果
- 提供一个简单的 CTA 按钮

## 文件结构

```text
ai-story-video-prompt-generator/
├── index.html
├── styles.css
├── app.js
└── README.md
```

## 文件说明

- `index.html`：页面结构，包含标题、介绍、表单、结果区域和 CTA 按钮。
- `styles.css`：页面样式，让网站更整洁、美观，并支持手机屏幕。
- `app.js`：页面交互逻辑，负责读取用户输入并生成提示词。
- `README.md`：项目说明文档。

## 如何使用

直接用浏览器打开 `index.html` 文件即可。
